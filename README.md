# 調整さん Googleカレンダーエクスポーター

調整さんのイベントページで、全員が◯の日付をGoogleカレンダーにエクスポートできるChrome拡張機能です。

## 機能

- 調整さんのイベントページで全員が◯の日付を自動検出
- 各日付を個別にGoogleカレンダーにエクスポート
- すべての日付を一度にエクスポート

## インストール方法

1. このフォルダをダウンロードまたはクローン
2. Chromeで `chrome://extensions/` を開く
3. 右上の「デベロッパーモード」を有効にする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. このフォルダを選択

## 使用方法

1. 調整さんのイベントページ（`https://chouseisan.com/s?h=...`）を開く
2. ページが読み込まれると、イベントヘッダーの下に「Googleカレンダーにエクスポート」セクションが表示されます
3. 各日付の「エクスポート」ボタンをクリックすると、Googleカレンダーのイベント作成ページが開きます
4. 「一括エクスポート」ボタンで、すべての日付をまとめてエクスポートできます

## 注意事項

- 日付の解析は自動で行われますが、形式が異なる場合は正しく動作しない可能性があります

### ファイル構成

- `manifest.json` - Chrome拡張機能の設定ファイル
- `content.js` - ページに注入されるスクリプト（メインロジック）
- `popup.html` / `popup.js` - 拡張機能のポップアップUI
- `styles.css` - エクスポートボタンのスタイル
